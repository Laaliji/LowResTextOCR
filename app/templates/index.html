Comment ça marche

Notre application utilise des techniques avancées de traitement d'images et de vision par ordinateur pour optimiser la reconnaissance de texte dans des images de faible qualité.

1

Acquisition d'image

Téléchargement et analyse

L'utilisateur télécharge une image contenant du texte. Le système analyse les propriétés de l'image : résolution, profondeur de couleur, et rapport signal/bruit.

Formats supportés: JPG, PNG Échantillonnage Quantification
2

Simulation de dégradation

Reproduction de défauts courants

Pour tester la robustesse de notre algorithme, nous simulons une image à faible résolution en ajoutant du bruit gaussien, en réduisant la résolution et en appliquant un flou.

Réduction de résolution Bruit gaussien Flou gaussien Compression JPEG
3

Prétraitement avancé

Techniques de restauration d'image

Nous appliquons une série de techniques avancées pour améliorer la qualité de l'image et rendre le texte plus visible et lisible pour l'algorithme OCR.

Extraction OCR

Reconnaissance de texte

Le moteur OCR Tesseract analyse l'image prétraitée pour identifier et extraire le texte. Notre application compare les résultats obtenus avant et après prétraitement pour démontrer l'efficacité des techniques appliquées.

À propos de cette application

OCRVision est une application développée dans le cadre d'un projet universitaire de Computer Vision. Elle utilise des techniques avancées de traitement d'images et l'OCR (Reconnaissance Optique de Caractères) pour extraire du texte à partir d'images à faible résolution.

L'application a été implémentée avec Python (Flask) pour le backend, OpenCV pour le traitement d'images, Tesseract pour l'OCR, et Tailwind CSS pour l'interface utilisateur. Elle démontre l'efficacité des techniques de prétraitement pour améliorer la qualité de l'extraction de texte.
